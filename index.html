Description

The CloudScale Showcase is a showcase for validation of CloudScale environment tools. It's a simple JavaEE web application for shopping a books written in SpringFramework. We also use the application to measure different metrics (e.g. scalability and elasticity) of application deployed in different environments. That includes public clouds (Amazon) and private clouds (OpenStack).

In order to deploy showcase on different cloud providers and measure different metrics, we also developed tools for that and are available on our GitHub account. These tools are actually deployment scripts for deploying showcase on different cloud providers and web application for deploying and executing different JMeter scenarios on Amazon Web Services.
