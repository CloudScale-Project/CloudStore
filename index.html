Description

The CloudScale Environment is an open-source application that integrates different tools for the analysis of cloud applications and their deployments, which are part of the CloudScale project.

As an example of how the CloudScale Environment allows us to migrate legacy systems or improve existing cloud applications, we created a the CloudStore application, based on the requirements and specifications of the TPC-W standard definition for an e-commerce site.

Phases

The progress of the implementation are divided into three incremental phases. Phase 0 is the legacy monolithic system, Phase 1 is a modernized version using current technologies for scalability, Phase 2 will provide with a fully and elastically scalable implementation, and the Phase 3 will be focused in the reduction of the operational costs of the entire system.
